• Welcome To the proof-of-concept of the Blockchain & Student Finance thesis by G. de Jonge. In order to launch the demo, please follow the upcomming steps.
  • Step 1: Install & Run TestRPC If not installed yet, install: Node.js
    Open the Windows Command Prompt (CMD).
    Click on Copy This , paste these commands in CMD and press enter.
    Do not close this CMD window.
  • Step 2: Preparing Remix IDE Go to: Remix IDE.
    Click on the Run tab at the top-right corner and change the Environment to Web3 Provider. Select http://localhost:8545 as the local host address.
    Click on the plus symbol (create new file) at the top-left corner and create a new file called StudentFinance.
    Click on Copy Code to copy the minified Solidity code to your clipboard. Now paste this in your StudentFinance file. The code can also be found on GitHub.
  • Step 3: Connecting Remix IDE Click on the Compile tab at the top-right corner and click on Start to Compile.
    Click on the Run tab at the top-right corner and click on Deploy. A new contract a will show up at the right.
    Click on the icon of your new contract to copy the contract address. Paste the address in the textbox below.
  • Proof-Of-Concept Oops! Something went wrong. Have you followed all of the previous steps? If so, please take the following actions:

    Shut down the CMD window(s).
    Close the REMIX IDE browser tab.
    Refresh this website, and try again.
